W przypadku problemow  z uruchomieniem mała instrukcja.

użyj wirtualnego środowiska (venv)
	1.	Utwórz środowisko:

python3 -m venv venv

	2.	Aktywuj środowisko:

source venv/bin/activate

	3.	Zainstaluj pakiety:

pip install openai
pip install pandas
pip install matplotlib
pip install tabula

	4.	Uruchom skrypt:

python model.py
